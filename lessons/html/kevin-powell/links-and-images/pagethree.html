JPGs

JPG images are good for photos.

GIFs

GIFs are only good for animations.

PNGs

PNGs are great for icons, logos, technical diagrams, etc. They are larger file sizes. Supports transparency.

SVGs

SVGs are vector-based. Great for responsive design - same use cases as PNG.

HTML Entities

There are three reserved characters in HTML: < > and &. You should always use HTML entities for these three characters.
